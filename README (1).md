# 课程视频质量分析工具

## 项目背景

在AI教育公司担任课程中台期间，需要从大量视频中筛选高质量内容、淘汰低效资源。原有流程依赖人工逐条查看数据，效率低且主观性强。本工具通过自动化分析视频观看行为数据，量化内容质量，辅助课程选品决策。

## 解决的问题

- 人工筛选效率低：数百条视频数据靠肉眼对比
- 判断标准不统一：缺乏量化的质量评估体系
- 报告输出费时：每次汇报需手动制图整理

## 功能介绍

- **自动评分**：基于认真观看率、完播率、观看量综合计算质量得分
- **内容分级**：自动标注优质/合格/待优化/低效四级标签
- **可视化报告**：生成散点图、饼图、Top10排行图
- **Excel导出**：输出多Sheet报告，含完整结果、优质清单、待优化清单

## 技术栈

- Python 3.x
- pandas（数据清洗与分析）
- matplotlib（数据可视化）
- openpyxl（Excel报告导出）

## 使用方法

### 安装依赖
```bash
pip install pandas matplotlib openpyxl
```

### 运行分析
```bash
# 方式一：命令行传入文件路径
python analyze.py your_data.xlsx

# 方式二：运行后手动输入路径
python analyze.py
```

### 数据格式要求

| 列名 | 说明 |
|------|------|
| 视频id | 视频唯一标识 |
| 视频名称 | 视频标题 |
| 总观看次数 | 点击播放总次数 |
| 认真观看次数 | 观看时长超过一定比例的次数 |
| 完播次数 | 看完整个视频的次数 |
| 认真观看率 | 认真观看次数/总观看次数 |
| 完播率 | 完播次数/总观看次数 |

### 输出结果

运行后在 `output_reports/` 目录生成：
- `analysis_chart.png`：可视化分析图表
- `video_analysis_report.xlsx`：详细分析报告（含4个Sheet）

## 评分算法

```
质量得分 = 认真观看率 × 40% + 完播率 × 40% + 观看量占比 × 20%
```

| 得分区间 | 评级 |
|---------|------|
| ≥ 70 | 🏆 优质内容 |
| 50-70 | ✅ 合格内容 |
| 30-50 | ⚠️ 待优化 |
| < 30 | ❌ 低效内容 |

## 项目意义

将原本需要30-60分钟的人工筛选工作，缩短至1分钟内自动完成，同时建立了可复用的量化评估标准。
